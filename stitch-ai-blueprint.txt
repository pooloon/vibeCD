【지원 국가 Reference Pack】
| 코드 | 국가     | 통화 | 기본 은퇴 | 절세·우대 계좌 (요약)              |
|------|----------|------|-----------|-------------------------------------|
| KOR  | 대한민국 | KRW  | 60세      | ISA, 연금저축, IRP                  |
| JPN  | 일본     | JPY  | 65세      | 신NISA, iDeCo, 特定口座             |

| GBR  | 영국     | GBP  | 66세      | S&S ISA, LISA, SIPP                 |
| SGP  | 싱가포르 | SGD  | 65세      | CPF, SRS                            |

【회원가입 2단계 플로우 — 화면 설계】
Step 1 · 계정

  - CTA: 「다음 — 국적·출생연도」

Step 2 · 국적·출생 (국가 선택 후 해당국 출생연도 목록 표시)
  - 국적 카드 그리드: 🇰🇷 🇯🇵 🇺🇸 🇬🇧 🇸🇬 (국기+라벨+코드)
  - 선택 시 Reference Pack 배너 슬라이드 인 (절세계좌·통화·기본 은퇴)
  - 출생연도 select: 국가별 라벨 (예: KOR 「1990년생 (만 36세)」)
  - 성별(선택), 거주지(선택)
  - CTA: 「가입 완료」

【로그인】
  - 이메일·비밀번호
  - 「가입 시 등록한 국적·출생연도가 자동 적용됩니다」

SECTION 3 — Security Engine (핵심 · v2 우선순위)
================================================================================

【목표 우선순위】
1. 종목 추천·순차 전략 (나이·위험성향·국적 → 코드×순서×비중)
2. 실시간(준) 시세·지표 (가격·PER·배당·52주·시총)
3. 이슈·미래가치 평가 (뉴스 톤 + 거시·섹터·밸류 → A~D)
4. 기대수익 Bear/Base/Bull (확률·가정 명시, 보장 없음)
5. 절세계좌·브로커 배치 (보조)

【데이터 소스】
| 데이터           | API/출처                          | 용도                    |
|------------------|-------------------------------------|-------------------------|
| KRX 전종목       | data.krx.co.kr (KOSPI+KOSDAQ)       | 시장·시세 탭 전체 목록  |
| Yahoo Finance    | 시세·PER·차트·백테스트              | 지표·6mo/1y/5y 차트     |
| Open DART        | opendart.fss.or.kr                  | 공시·재무제표·기업개요  |
| Yahoo News       | 종목별 헤드라인                     | 이슈 스코어 보조        |

【티커 규칙】
- KOSPI: 005930.KS | KOSDAQ: 035720.KQ | 일본: 7203.T | 미국: AAPL, SPY

【이슈 등급 A~D】
- A: 코어·위성 적합 (분산·DCA)
- B: 비중 제한·분할 매수
- C: 투기적 소액만
- D: 회피 또는 극소 비중
+ 뉴스 톤: 우호 / 중립 / 부정

【공통 헤더】
- Eyebrow: 「Security Engine · DART · PDF · Alerts」
- H1: 「종목·시세·순차전략 투자 설계」
- 프로필 바: 회원명 | 국적·출신 (국기+코드) | 나이 (만 N세·N년생) | 로그아웃

- Tab Nav (pill): 순차 전략 | 시장·시세 | 백테스트 | 통합 리포트 | 알림 | 재무·설정

──────────────────────────────────────────────────────────────
TAB 1 · 순차 전략
──────────────────────────────────────────────────────────────
- Executive Summary (국적·나이·위험·은퇴·Base case 연 X% 참고)
- 자산 배분 chips: 주식·ETF % | 채권 % | 금 % | 위성 %
- 순차 매수 로드맵 테이블:
  # | Phase | 기간 | 종목 | 상장코드 | 비중 | 계좌(ISA 등) | 액션 | 등급
- 기대수익 시나리오 카드 3개: Bear / Base / Bull
  - 연 %, 참고 확률, 적립 시 terminal 금액(입력 시)
- 면책 bullet list

──────────────────────────────────────────────────────────────
TAB 2 · 시장·시세 (KRX · DART · 검색)
──────────────────────────────────────────────────────────────

- 버튼: KRX 새로고침 | 뉴스
- 필터: 검색(종목명·코드·섹터) + 자산군(전체/주식/ETF/채권ETF/원자재)
- 페이지네이션 (80건/页)
- 테이블: 종목 | 코드 | 시장 | 현재가 | 등락 | PER | 시총 | 등급 | 공시 | 상세
- 행 클릭 → 종목 상세 (아래)
- KOR: 우측 DART 공시 패널 (sticky) — 공시 목록 + DART 링크

【종목 상세 서브화면】← 「차트·재무」
- ← 목록 | 종목명 + 코드 | 등급 badge
- Metric cards: 현재가 | PER | 기간 수익률 | [목표가 알림] 버튼
- 차트 range tabs: 6개월 | 1년 | 5년 (SVG 라인 차트)
- 이슈·전망 (거시·섹터·밸류 bullet)
- DART 재무제표 테이블: 계정 | 당기 | 전기
  (매출·영업이익·당기순이익·자산·부채·자본·영영CF)
- 최근 공시 리스트

TAB 3 · 백테스트

- 테이블: 종목 | 코드 | 기간 | CAGR | 총수익 | MDD | 연변동성
- 면책: 과거≠미래, 수수료·배당 미반영

──────────────────────────────────────────────────────────────
TAB 4 · 통합 리포트 (Step 0~9)
──────────────────────────────────────────────────────────────

- Markdown 형식 프리뷰 (스크롤):
  0. 프로필 & Reference Pack

SECTION 5 — Sequential Strategy (순차 전략 로직)
================================================================================

【Phase 로드맵】
| Phase       | 시점      | 내용                                      |
|-------------|-----------|-------------------------------------------|
| 1 Core      | 1~6개월   | 코어 인덱스 ETF — ISA/NISA 등 절세계좌 DCA |
| 2 Defensive | 7~12개월  | 국채·채권 ETF + 금 ETF 헤지               |
| 3 Satellite | 13개월~   | 이슈 A/B 위성 종목 분할 매수              |
| 4 Glide     | 은퇴 10년~| 주식→채권 리밸런싱 (연 5%p)               |

【KOR 우선순위 (조정 가능)】
1. ISA → 성장·배당·금 ETF (비과세 한도)
2. 연금저축 + IRP → 세액공제
3. 앱금융 vs 전통사 → 수수료·환전·ISA 연계
4. 일반 위탁 → 한도 초과·단기·일부 BTC

【기대수익 시나리오 (참고용 · 보장 없음)】
- Bear:  ~20% 확률 가정, 침체·고금리
- Base:  ~55% 확률, 장기 평균 근처
- Bull:  ~25% 확률, 저금리·서프라이즈
※ 월 적립액 입력 시 복리+적립 terminal 금액 산출
