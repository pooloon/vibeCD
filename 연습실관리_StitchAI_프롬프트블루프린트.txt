■ 이 문서 용도
Google Stitch AI에 화면(UI) 디자인을 생성·개선할 때 복사해 넣는 프롬프트 모음입니다.
코드 구현은 별도(React 웹 / Kotlin Android)이며, Stitch에서는 **시각 디자인·UX 와이어**에 집중합니다.

2. Stitch AI 공통 설정 (매 세션 시작 시 붙여넣기)
================================================================================

---[ Stitch 시스템 프롬프트 / 프로젝트 설정 ]---

당신은 소규모 연습실·스튜디오 운영 SaaS가 아닌, **1인 운영자용 실무 앱** UI를 설계합니다.

디자인 원칙:
- 화려한 마케팅 랜딩이 아니라 **업무용 도구** 느낌
- 입력은 **3탭 이내**로 끝나게 (필드 최소화, 큰 터치 영역)
- 숫자(금액·건수)가 한눈에 보이는 **KPI 카드** 우선
- 초보 운영자도 이해 가능한 **직관적 한국어 라벨**
- 오프라인·저조도 환경 고려 (대비 충분, 작은 글씨 지양)

타겟 디바이스:
- Primary: Android 스마트폰 (360×800 ~ 412×915)
- Secondary: PC 브라우저 (최대 너비 720px 중앙 정렬 느낌)

디자인 토큰 (기존 웹앱과 통일):
- 배경: #F4F6F9
- 서피스(카드): #FFFFFF
- Primary(네이비): #1E3A5F
- Primary Light: #2D5A8E
- 수익(틸): #0F766E / 배경 #CCFBF1
- 지출(앰버): #B45309 / 배경 #FFEDD5
- 미실현·보조: #64748B
- 위험·취소: #DC2626
- 테두리: #E2E8F0
- 모서리: 14px 라운드
- 그림자: 부드러운 카드 섀도우
- 폰트: Pretendard 또는 시스템 산세리프

네비게이션:
- 하단 고정 Bottom Navigation (5탭)
- 아이콘 + 짧은 한국어 라벨
- 활성 탭은 Primary 색 강조

금액 표시: 원(KRW), 천 단위 콤마 (예: 350,000원)
날짜: 한국 로케일 (예: 2026년 7월 4일, 7/4 14:00)

하지 말 것:
- 영문 UI 기본 제공
- 로그인·회원가입·소셜 로그인 화면 (단일 운영자, 인증 없음)
- 과도한 일러스트·3D·그라데이션 남용
- 서버 연동·클라우드 동기화 UI

스타일: 실무용, 깔끔, 카드 기반, Primary #1E3A5F, 배경 #F4F6F9, 한국어 전용

모바일 퍼스트 연습실 운영 관리 웹앱 UI 키트를 디자인해 주세요.

다음 5개 화면을 동일한 디자인 시스템으로 생성:

모든 화면에 하단 5탭 네비게이션 포함.
